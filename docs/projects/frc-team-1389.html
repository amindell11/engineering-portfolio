FIRST Robotics

Overview

As a software developer for FIRST Robotics Competition Team 1389 (STRATUS), I contributed to building a comprehensive software ecosystem for competitive robotics. The work spanned autonomous path-following algorithms, computer vision systems for target detection, reusable software libraries, and real-time web dashboards for monitoring robot telemetry during matches.

Key Contributions

  • Autonomous Path-Following: Developed algorithms enabling the robot to execute pre-planned trajectories during the autonomous period of FRC matches with high precision and repeatability.
  • Computer Vision: Implemented vision pipelines for real-time target detection and tracking, enabling automated aiming and alignment with field elements.
  • Reusable Libraries: Built modular, well-documented software libraries that could be reused across seasons and shared with other teams.
  • Real-Time Web Dashboards: Created web-based dashboards using WebSocket connections for live telemetry monitoring, enabling the drive team to make data-driven decisions during matches.
