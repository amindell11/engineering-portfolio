IDT Winter 2016 Competition — Team "The Flatlanders"

Competition
Winter 2016

Overview

Built a geolocation-based package tracking web application in under 48 hours for the IDT Winter 2016 Coding Contest as part of team "The Flatlanders." The app shipped with real-time map visualization, UUID-based package tracking, ETA estimation, and delivery status management — evaluated by industry judges on correctness, efficiency, and presentation.

Groovy Package Tracker

Designed and implemented the full-stack application in Groovy with a web frontend backed by geolocation APIs:

  • Real-time map interface: Visualized package routes between cities on an interactive map with live position updates.
  • UUID tracking & ETA: Each package received a unique identifier with computed distance and estimated arrival time based on routing data.
  • Delivery management: CRUD operations for packages, notes, and status tracking with persistent storage.
