Aurora Flight Sciences

Software Intern — Embedded Collision Avoidance for Autonomous UAVs

Internship
Summer 2016
Cambridge, MA

Overview

Cut obstacle reaction latency by 80% (150 ms → 30 ms) and reduced verification cycle time by 15% during a summer internship at Aurora Flight Sciences (now a Boeing company). I owned the end-to-end integration of an ultrasonic collision-avoidance module for autonomous quadrotor UAVs, working under flight-critical real-time constraints.

Collision Avoidance Integration

Delivered a fully integrated ultrasonic sensing pipeline — from raw serial data to flight-controller commands — that met strict real-time deadlines on every test flight:

  • 80% latency reduction: Refactored the C++ I/O pipeline to cut reaction time from 150 ms to 30 ms with ≤ ±2 ms jitter, by eliminating redundant data copies and streamlining the sensor-to-actuator path.
  • Full-stack sensor integration: Implemented low-level serial communication, data parsing, filtering, and obstacle-distance injection into the flight controller's avoidance logic.
  • Flight-critical reliability: Validated timing guarantees under worst-case load to ensure no missed deadlines during autonomous flight.

Test & Verification

Authored formal test procedures and execution checklists that became the team's standard for collision-avoidance verification:

  • 15% faster verification cycles: Structured, repeatable procedures replaced ad-hoc testing, cutting the time from code change to validated result.
  • Institutional documentation: Checklists remained in use by the team after the internship ended, improving consistency across subsequent development sprints.
